Installing Scala

For resources on getting started programming in Scala, have a look at the Getting Started guide. First, here are the most popular ways to install Scala on your machine:
  • Download the Scala distribution, unpack the archive, and launch scala (in the bin folder). This starts the interactive Scala interpreter (aka the "REPL"). The standard batch compiler can be invoked from the command line as scalac. We recommend one of the build tools that support Scala:
    • sbt,
    • maven with the Scala plugin,
    • or the Scala plugin for gradle.
  • Install one of the Scala IDEs: the Scala IDE (based on Eclipse), IntelliJ IDEA with the Scala plugin, or NetBeans IDE with the Scala plugin
    • The Scala IDE and IntelliJ IDEA both support Scala worksheets, interactive Scala documents that are continuously evaluated line-by-line. Worksheets are an excellent way to experiment with Scala while still being able to save your code as in file and reuse it later on.
  • Typesafe Activator is a one-click way to get started creating applications with Scala, Akka and Play Framework. Activator comes with dozens of templates to help you get started quickly. Create your own template to help programmers discover and quickly get up to speed with your own project, or contribute to Activator itself!
